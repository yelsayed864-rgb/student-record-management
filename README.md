# Student Record Management System (C++)

A menu-driven C++ application designed to manage student records including IDs, grades, and statistical calculations.

## Features
- Add and store student records
- Display all student information
- Calculate class average
- Menu-driven user interaction
- Input validation to prevent invalid entries

## Concepts Used
- Vectors (dynamic arrays)
- Loops and conditionals
- Functions and modular design
- Basic statistics (average calculation)

## How to Run

Compile:
g++ main.cpp -o student_records


Run:


./student_records
